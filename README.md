# Me Inc - Coming Soon

A modern, responsive coming soon page for Me Inc.

## Project Structure

```
meincdotapp/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Custom styles
├── js/
│   └── script.js       # JavaScript functionality
├── assets/
│   └── images/
│       └── logo.jpeg   # logo
└── README.md           # This file
```

## Features

- **Responsive Design**: Works on all devices
- **Dark/Light Mode**: Automatically detects system preference
- **Modern UI**: Clean design
- **Optimized**: Fast loading with minimal dependencies

## Setup

Simply open `index.html` in any modern web browser. The page will automatically adapt to your system's theme preference.

## Technologies Used

- HTML5
- CSS3 (with Pico.css framework)
- Vanilla JavaScript
- Responsive design principles
